void IPlugBase::ProcessDoubleReplacing ( double **  inputs,
double **  outputs,
int  nFrames 
)
virtual

Default passthrough.

Inputs and outputs are two-dimensional arrays [nChannel][nSample]

Parameters
inputs2D array containing the input buffer for all channels. To access a specific channel:
double* inLeft = inputs[0];
double* inRight = inputs[1];
The result is a double array, indexed from 0 to nFrames which can be accessed in this manner:
for (int i = 0; i < nFrames; ++i) {
double sampleL = inLeft[i];
double sampleR = inRight[i];
}
For multi-channel plugins, use subsequent index numbers
outputs2D output array. The output of your DSP goes here
nFramesNumber of samples per array (buffer size)
